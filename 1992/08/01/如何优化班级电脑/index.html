如何优化班级电脑

如何优化班级电脑

注: 本文原打算投稿南九校刊, 然后说是会被砍掉, 就改成投稿到这.

(如何使配置贼差的班级电脑运行的更流畅)

Warning: 本文目的是使班上的电脑更好用, 更流畅. 因各班电脑一定或多或少有些许差异, 故具体如何优化请参照本班电脑实际情况决定. (总之就是出了问题别找我)

本文建立在以下条件下:

  • 你是班上计算机管理员
  • 电脑操作系统是 Windows 7
  • D盘不受影子系统保护

本文将从以下几点进行优化:

  • 文件系统
    • 桌面位置
    • 文件整理
    • 磁盘管理
  • 办公软件
    • Office
    • WPS
    • Adobe Acrobat / Edge
  • 系统软件
    • IE
    • Flash
    • 运行库
  • 浏览器
    • Edge / Chrome
      • 插件
        • 油猴
        • AdBlock
  • 其他应用程序
    • 希沃
      • 希沃PPT小工具
      • 希沃管家
      • 希沃浏览器
      • 希沃轻白板
    • 音视频
      • PotPlayer / VLC
    • 图片
      • Windows 图片查看器
    • 压缩文件
      • 7z / Bandizip
  • 润物细无声的小工具
    • 自动打开U盘
    • 桌面壁纸添加当日课程表
    • 等等
  • 计算机操作系统 (危)
  • 一些小提示

Warning: 请在时间充裕的情况下操作, 否则你必须有十足的把握.

0x00 禁用影子系统

要优化它, 第一步肯定就要把影子系统先暂时禁用, 不然做的就都是无用功了.

禁用 (或绕过) 影子系统的方法有三种:

  • 输入正确的密码
  • 开机选择模式时输入一串神秘字符
  • Win PE

第一种的话, 密码 123456, 000000 这些自己试试, 总有一款适合你, 我们班密码是前者. (建议使用本方法);

第二种的话, 在开机选择进入正常模式还是影子模式的时候选择前者, 然后输入280左右个数字”8”, 亲测有几率解开;

第三种需要带实体键盘和装有 PE 系统的 8GB 以上U盘来学校, 故不考虑 (虽然我也会用到).

还有一种比较直接的办法: 进安全模式或PE直接把影子系统干掉. 不建议, 鬼知道班上电脑会运行什么程序, 一会中病毒上课时电脑用不了老师要找你麻烦的.

0x01 文件系统

  1. 桌面位置

    首先肯定是先建议把桌面位置挪到D盘等不受影子系统保护的盘里, 这样老师往桌面拷东西并声称下节课还要用时, 你就不用辛辛苦苦下课时再把桌面的课件移到D盘以防被影子系统还原了.

    操作方法:

    1. 打开桌面上的”计算机” (Windows 资源管理器 / 文件资源管理器);
    2. 打开D盘或其他不受影子系统保护的盘 (以D盘为例);
    3. 空白位置右键并点击新建文件夹 (文件夹名以”Desktop”为例)
    4. 在左侧的导航窗格找到”桌面”;
    5. 右键, 点击属性;
    6. 复制”目标 (T)”旁的路径;
    7. 关掉”属性”窗口, 回到”文件资源管理器”窗口;
    8. 在上方的地址栏粘贴刚才复制的路径并回车;
    9. 在空白位置右键, 点击属性;
    10. 在属性窗口点击上方的”位置”Tag;
    11. 点击”移动”按钮, 选择D盘下的Desktop文件夹, 点击”选择文件夹”;
    12. 点击”应用”;
    13. 点击”是”;
    14. 完成.

    (再也不写这么长这么详细了, 累)

  2. 文件整理

    建议在D盘根目录下 (方便老师寻找) 新建多个文件夹, 对应存放各个学科老师的资料, 免得搞混难找.

    可以在桌面创建对应快捷方式.

  3. 磁盘管理

    一般来说班上电脑E盘是没用的, 可以将其在磁盘管理或DG中删除盘符号以隐藏 (防止老师找课件找错盘) 或将其格式化并与D盘合并 (出了问题别找我).

0x02 办公软件

办公软件主要就是用来打开老师的课件的, 为了兼容性建议 WPS 和 Office 都装, 以防出现兼容性问题 (如经典的 WPS 中编辑的课件到 Office 中文本框位置会偏移等). 我们班用的是 Office 2016 (太新可能会出问题) + WPS 2019 教考版 (不用登陆, 没广告)

PDF格式的话主要就是放各种联考的答案用, Edge 就行, 但我们班用的是 Adobe Acrobat.

0x03 系统软件

这个主要是 IE 和 Flash. 要用到 Flash 的话可以装个360浏览器之类的国产浏览器, 一般都会支持 Flash (现在哪个网站还用 Flash 的话记得去踢它前端两脚); IE 的话建议直接在控制面板禁用, 免得它总把自己当成默认程序 (现在哪个网站还要求 IE 访问的话记得去踢它前端两脚 (不记得我在哪个网站前端源码 (F12) 看到过一行注释: “IE must go die!”, 前端狂喜了属于是) ), 如果真要用的话就用 Edge 浏览器的 IE 模式.

运行库的话还是有必要装一下的, 不然哪天老师的软件 (如我们英语老师的抽学号软件) 用不了还得怪你把电脑整坏了.

0x04 浏览器

Edge 和 Chrome 任选其一, 都挺好用的, 建议 Edge, 毕竟还有个 IE 兼容模式.

插件嘛, 肯定得装两个, 新生开学不久应该就有国歌校歌比赛, 这时候肯定会用到浏览器搜音乐, 不装几个插件就广告漫天飞了. 建议装油猴 (用于安装各种脚本, 可精准优化各个网站得使用体验 (我也有写过脚本) ) 和 AdBlock (用于屏蔽网站的广告). 都非常好用.

0x05 其他应用程序

首当其冲的肯定是希沃全家桶, 建议直接卸载以下几个软件:

  • 希沃浏览器 (Edge 不香吗)
  • 希沃轻白板 (希沃白板觉得很赞)
  • 希沃管家 (啥用没有还看着碍眼)

不过”希沃PPT小工具”还是挺好用的, 如果班上电脑没装的话建议去希沃官网 (e.seewo.com) 下一个装.

音视频方面建议用 PotPlayer 或 VLC, 我们班用的是前者, 不过记得去设置里把自动更新和新窗口打开文件关掉.

图片用 Windows 图片查看器 足矣, 我个人用的是 ImageGlass, 挺好用的.

压缩文件的话 7z 和 Bandizip 任选其一, 我们班用的是前者, 我个人用的是后者.

0x06 润物细无声的小工具

自己写了几个小工具, 一个是当U盘插入时自动打开U盘 (文件资源管理器); 另一个则每天自动把桌面壁纸换成 Bing 每日一图并附带当天课程表 (如果要自己实现这个小工具的话注意不要把课表在程序源码里写死, 建议外存为 JSON 或 ini 文件以防临时调课); 还有一个就是自动在上课前把对应学科存在D盘的课件剪切到桌面, 但后来因为时不时的换课, 不方便用而被我删了; 还有一堆懒得打字了…

0x07 计算机操作系统 (危)

Warning: 高级操作, 不建议尝试!

Warning: 重装系统有风险, 谨慎操作!

Warning: 时间充裕时方可操作!

Warning: 刷坏了别找我!

可以把希沃自带的 Win 7 刷到 Win 10 或 Win 11. 已有成功案例 (启智楼502和604, 分别刷了10和11).

注意要先备份一些数据, 如电脑 IP, 网关等.

一定要备好一个实体键盘和一个装有 PE 系统的 8GB 以上U盘, PE 系统建议 微PE (不能联网) 或 FirPE (可联网). 实体键盘用于进 BIOS 和 BOOT 列表; PE 用于装系统或在装坏时补救.

安装完后记得去希沃官网下载对应驱动装上, 建议同时把”希沃白板”和”希沃视频展台”一起装上, 老师们挺常用的.

0x08 一些小提示

  1. 有些电脑可能会出现 QQ 窗口无法移动的问题, 打开右下角的”TouchService”, 把”多点触控” (好像是这个名字) 关掉就好.
  2. QQ 的滑动验证码划不过去: 同上, 如果还不行的话, 就同时在滑动验证码滑块起点和终点位置点击, 也有几率通过. 老师的话建议使用扫码登录, 方便快捷.
  3. 想不出来了, 应该凑够字数了吧… (实际上已经3k了)
